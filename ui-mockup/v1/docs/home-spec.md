# ホーム画面仕様 (Mobile) - v1

## ボトムナビゲーションタブ（全5つ）

| # | key | label | icon | 遷移先 |
|---|-----|-------|------|--------|
| 1 | home | ホーム | ⌂ | home.html |
| 2 | stats | 統計 | 📊 | stats.html |
| 3 | tools | ツール | 🔧 | tools.html |
| 4 | wallet | ウォレット | 👛 | wallet.html |
| 5 | menu | メニュー | ≡ | menu.html |

※ 履歴・単体申請・資料などは各ページから遷移する別ページ

---

## ホーム画面 構成要素（上から順）

### 1. ヘッダーバー
| 要素 | 内容 | 配置 |
|------|------|------|
| アプリ名 | `THEME.app_name`（ゆうき銀行） | 左 |
| アカウント名 | `THEME.user_main_name`（ゆうき） | 右 |
| アカウントアイコン | 頭文字アバター（ゆ） | 右端 |

---

### 2. メイン残高エリア
| 要素 | 内容 |
|------|------|
| ラベル | `THEME.home_main_balance_label`（メイン残高） |
| 金額 | 大きく中央表示（例: ¥12,500） |

---

### 3. クイックアクション（ナビゲーションボタン）
横並び4つ、アイコン＋ラベル

| # | key | label | icon | 動作 |
|---|-----|-------|------|------|
| 1 | charge | チャージ | ↑ | チャージ申請画面へ |
| 2 | withdraw | 引き出し | ↓ | 引き出し申請画面へ |
| 3 | payment | 建て替え | ⇄ | 建て替え申請画面へ |
| 4 | other | その他 | ⋯ | その他メニュー展開 |

---

### 4. タクソノミー残高ボタン（もちポイント / NISA / その他）
横スクロール可能なカード群、タップで各タクソノミー詳細画面へ遷移

| # | 表示名 | 残高例 | 単位 | タップ時 |
|---|--------|--------|------|----------|
| 1 | もちポイント | 1,200 | pt | ポイント詳細へ |
| 2 | NISA | ¥50,000 | 円 | NISA詳細へ |
| 3 | ガチャコイン | 350 | 枚 | コイン詳細へ |

※ タクソノミーの数は可変（データから動的生成）
※ タップ先は別ページ（後日作成）

---

### 5. 承認待ちセクション
| 要素 | 内容 |
|------|------|
| セクションタイトル | 「承認待ち」＋件数バッジ |
| 表示件数 | **最大3件** |
| 各アイテム表示 | アイコン / ラベル / カテゴリ・日付 / 金額（±色分け） |
| 3件超の場合 | 「すべての承認待ちを見る」ボタン表示 |
| 0件の場合 | セクション非表示 or 「承認待ちはありません」 |

アイテム1行の構成:
```
[icon] [ラベル              ] [金額(色付き)]
       [カテゴリ・日付(灰)]
```

---

### 6. 最近の取引セクション
| 要素 | 内容 |
|------|------|
| セクションタイトル | 「最近の取引」 |
| 表示件数 | **最大10件** |
| 各アイテム表示 | アイコン / ラベル / カテゴリ・日付 / 金額（±色分け） |
| 下部 | 「すべて見る」ボタン → 履歴ページへ |
| 0件の場合 | 「取引履歴はありません」 |

アイテム1行の構成: 承認待ちと同一フォーマット

---

### 7. ボトムナビゲーション（固定）
画面下部に固定表示、5タブ（上記タブ定義参照）
- アクティブタブはアクセントカラーで強調
- バッジ: 承認待ち件数をホームタブに表示可能（検討中）

---
