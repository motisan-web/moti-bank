# 統計ページ仕様 (Mobile) - v1

## ページ構成（上から順）

---

### 1. ページヘッダー
| 要素 | 内容 |
|------|------|
| タイトル | 「統計」 |
| 期間切替タブ | 1ヶ月 / 3ヶ月 / 6ヶ月 / 1年 / 全期間 |

※ 期間タブは一部のグラフ・数値に影響（対象データの範囲が変わる）

---

### 2. 今月の建て替え（棒グラフ）【必須】
| 要素 | 内容 |
|------|------|
| タイトル | 「建て替え推移」 |
| グラフ種別 | 棒グラフ（縦棒） |
| X軸 | 日付（日単位 or 週単位、期間タブに連動） |
| Y軸 | 金額（円） |
| 棒の色 | サブカテゴリ別に色分け（food/gadget/office/daily/transport/other） |
| 下部 | 合計金額を表示 |

---

### 3. 残高の内訳（ドーナツチャート）【必須】
| 要素 | 内容 |
|------|------|
| タイトル | 「残高の内訳」 |
| グラフ種別 | ドーナツチャート |
| 中央 | 総残高（メイン + 全タクソノミー合算の円換算） |
| セグメント | メイン残高 / もちポイント / NISA / ガチャコイン |
| 凡例 | 各セグメント名 + 金額 + 割合% |

---

### 4. ポイント残高推移（折れ線グラフ）【必須】
| 要素 | 内容 |
|------|------|
| タイトル | 「ポイント・コイン推移」 |
| グラフ種別 | 折れ線グラフ（複数線） |
| 線 | もちポイント / ガチャコイン（各色分け） |
| X軸 | 日付（期間タブに連動） |
| Y軸 | 残高数 |
| タップ | 各点タップで詳細ツールチップ |

---

### 5. サマリーカード（数値ハイライト）
横スクロール or 2列グリッドのカード群

| # | カード名 | 表示値 | 説明 |
|---|---------|--------|------|
| 1 | 合計建て替え額 | ¥XX,XXX | 選択期間内の建て替え合計 |
| 2 | 合計チャージ額 | ¥XX,XXX | 選択期間内のチャージ合計 |
| 3 | 合計引き出し額 | ¥XX,XXX | 選択期間内の引き出し合計 |
| 4 | 純増減 | ±¥XX,XXX | 期間内の残高変動（plus合計 - minus合計） |

---

### 6. 年別サマリー（テーブル or 横棒グラフ）【必須】
| 要素 | 内容 |
|------|------|
| タイトル | 「年別サマリー」 |
| 表示形式 | 横棒グラフ or テーブル |
| 行 | 年（2026, 2025, ...） |
| 列 | チャージ合計 / 建て替え合計 / 引き出し合計 / 純増減 |

---

### 7. カテゴリ別内訳（横棒グラフ）
| 要素 | 内容 |
|------|------|
| タイトル | 「カテゴリ別支出」 |
| グラフ種別 | 横棒グラフ（ランキング形式） |
| 項目 | 建て替えサブカテゴリ別（food/gadget/office/daily/transport/other） |
| ソート | 金額の多い順 |
| 表示 | 棒 + 金額 + 割合% |

※ 選択期間に連動

---

### 8. 月別トレンド（折れ線 or エリアチャート）
| 要素 | 内容 |
|------|------|
| タイトル | 「月別収支トレンド」 |
| グラフ種別 | エリアチャート（plus / minus 重ね） |
| X軸 | 月（直近12ヶ月 or 期間タブ連動） |
| Y軸 | 金額 |
| 線 | 入金（plus系）/ 出金（minus系） |

---

### 9. 建て替え頻度ヒートマップ
| 要素 | 内容 |
|------|------|
| タイトル | 「建て替え頻度」 |
| 表示形式 | GitHub風カレンダーヒートマップ |
| セル | 日ごとの建て替え回数を色の濃淡で表示 |
| 範囲 | 直近3ヶ月 |

※ どの曜日・時期に建て替えが多いかが一目でわかる

---

### 10. ボトムナビゲーション（固定・共通コンポーネント）

```html
<div data-component="bottom-nav" data-active="stats"></div>
```

---

## グラフライブラリ

モック段階では **Chart.js**（CDN）を使用
- 軽量・レスポンシブ対応
- 棒・折れ線・ドーナツ・エリアチャート全対応
- `<canvas>` ベースで静的HTMLでも動作

---

## 非表示・後日対応
- グラフのドリルダウン（棒クリック → 該当取引一覧）
- データエクスポート（CSV）
- カスタム期間指定
- PC版の2カラムレイアウト
